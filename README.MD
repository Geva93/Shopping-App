# Shpping App (iOS & Android)

## Setup
To run this project, install it locally using npm:

```
$ cd {project-folder}
$ npm install
$ npm start
```

## Tools
Project is created with:
Redux
Remote DB
Navigation
Multiple Screens
and More...

*Before using, connect fetch functions to a working API
